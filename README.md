## Install
To install `snippets` use `go get`:
```shell
go get -u github.com/avnovoselov/snippets/issues
```

### Git
#### Remove tracking branches no longer on remote
Run in project directory
```shell
snippets git remove-orphans
```
or shortest
```shell
snippets git ro
```
